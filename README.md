# Vanguard TD

Basit ama taktik odaklı bir kule savunma oyunu demosu. Oyunu çalıştırmak için `index.html` dosyasını tarayıcıda açmanız yeterli.

## Oynanış
- **Kule Yerleştir** butonu ile boş bir kareye tıklayarak kule kurarsın.
- **Dalga Başlat** ile düşmanları gönderirsin.
- Düşmanlar üsse ulaşırsa can kaybedersin, kuleleri güçlendirmek için altın kazanırsın.
